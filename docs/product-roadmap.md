# FinGraph Product Roadmap

## Phase 1: Production Skeleton

- Next.js dashboard
- Supabase schema
- Source registry
- Seed data fallback
- Exportable Skill Pack
- Vercel Cron endpoint

## Phase 2: Evidence Warehouse

- Run collectors daily
- Store events and indicators
- Add source health checks
- Add duplicate detection by URL hash
- Add event confidence scoring

## Phase 3: Graph Intelligence

- Auto-map events to layers and nodes
- Add graph edge scoring
- Add historical layer score trends
- Add watchlist assets

## Phase 4: Report Workflow

- Export configurable Skill Packs
- Export weekly and monthly packs
- Save historical packs
- Add user notes and manual links

## Phase 5: Advanced Analysis

- Optional LLM-assisted classification
- Cross-source contradiction detection
- Portfolio-specific watchlists
- Alerts for rates, energy, filings, and geopolitical clusters
